[color=blue][b]Назначение проекта: [/b][/color][list]Предоставление доступа к локальной системе Windows путём смены пароля и статуса учётных записей.[/list]
[br]
[color=blue][b]Представляет собой: [/b][/color][list]Командный бат-файл.[/list]
[br]
[color=blue][b]Способ применения: [/b][/color][list]
Загружаемся из любой WinPE. Запускаем бат-ник, который:
[list][*]Просканирует диски в поисках Windows (если дисков с Windows не один, то предложит ввести нужный).
[*]Сделает твик реестра (куст System) и добавит бат-ник в корень найденного/выбранного системного диска.
[/list]
После чего можно перезагрузиться в основную систему. Перед окном входа появится консольное окно со списком пользователей. Ввести номер нужного пользователя и:
[list][*]Сменить ему пароль
[*]Активировать/Деактивировать
[*]Сделать локальным Администратором
[/list]
По завершении работы бат-ник удалит из реестра свой запуск и самого себя с системного диска.[/list]
[br]
Существует возможность автоматического создания локального Администратора, если номером пользователя ввести [b]544[/b].
Создаётся пользователь: [b]aaddmm66[/b] (если не введено другое имя) с паролем: [b]AdminPass666[/b]
[br]
[color=red][b]Важно![/b][/color] Если компьютер вошёл в "круговую перезагрузку", выполнить следующее:
Загрузиться в WinPE, выполнить бат-ник, согласиться с установкой обработчика паролей и затем выбрать "Удалить и выйти" (см. первый [i]screenshot[/i]).
